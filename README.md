 Personal Portfolio Website

A fully responsive portfolio website built using React.js, Tailwind CSS, and Framer Motion.

 Features

Built with React.js for component-based structure
Tailwind CSS for fast and responsive styling
Framer Motion animations for smooth transitions and interactions
Projects section with GitHub/live demo links
Contact section 

 Tech Stack

Frontend: React.js 
Styling: Tailwind CSS
Animations: Framer Motion
Icons: React Icons / Lucide Icons (update if applicable)
Deployment: Vercel 


 Getting Started

 1. Clone the repository

git clone https://github.com/yourusername/your-portfolio.git
cd your-portfolio


 2. Install dependencies

npm install


3. Run the development server

npm run dev


Live at : https://ramyaportfolio-pi.vercel.app
